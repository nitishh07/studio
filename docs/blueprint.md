# **App Name**: SwiftJobs

## Core Features:

- Job Listing Display: Show a paginated list of job postings on the homepage, displaying title, company, location, and job type. Job listings will update in real-time from Firestore.
- Job Search & Filter: Provide a search bar and filter options (e.g., job type, location) to refine job listings client-side.
- Detailed Job View: Dedicated page to display the full job description and an 'Apply Now' link for individual job postings.
- Admin Authentication: Secure admin login using Firebase Authentication with email and password.
- Job Management Console: An administrative interface for authenticated users to add, edit, and delete job postings stored in Firestore.
- AI Job Description Assistant: A generative AI tool within the admin panel to assist with crafting or enhancing job descriptions by suggesting content based on job titles and keywords.

## Style Guidelines:

- Primary color: A professional and trustworthy medium blue (#2258d4), chosen for clarity and reliability, creating good contrast on light backgrounds.
- Background color: A very light, slightly desaturated blue-gray (#eceff5), providing a clean and subtle backdrop that maintains the app's overall calm and professional aesthetic.
- Accent color: A vibrant, clear blue-green (#42b2cf), analogous to the primary color, used for interactive elements and highlights to ensure visibility and draw attention without being overwhelming.
- All text (headlines and body): 'Inter', a modern sans-serif typeface, selected for its objective, neutral, and highly readable qualities, fitting the 'simple and clean' requirement of a job board.
